# Privacy Policy

**Last updated: June 2026**

## Summary

This dashboard collects **no personal data whatsoever**. There are no accounts, no cookies, no trackers, and no user data of any kind — not even anonymised analytics.

---

## What This Dashboard Does

The India El Niño Intelligence Dashboard is a **static, read-only climate information portal**. It displays publicly available climate indices and environmental data from government meteorological agencies. Users visit the site, view charts, and leave. Nothing else happens.

---

## Data Collected: None

| Category | Collected? | Details |
|----------|-----------|---------|
| Personal identifiers (name, email, IP) | ❌ No | Not collected, not logged |
| Location data | ❌ No | Not requested, not inferred |
| Device / browser fingerprint | ❌ No | No tracking scripts |
| Cookies | ❌ No | No cookies of any kind |
| Analytics / usage telemetry | ❌ No | No Google Analytics, no Plausible, no Mixpanel |
| Health or medical data (PHI) | ❌ No | Not applicable |
| Financial data | ❌ No | Not applicable |
| User accounts / login | ❌ No | No authentication system exists |

---

## Data Displayed on the Dashboard

All data shown is **public-domain environmental data** from government agencies:

| Dataset | Source | Nature |
|---------|--------|--------|
| Niño3.4 / ONI | NOAA/CPC (USA) | Sea surface temperature averages |
| SOI | NOAA CPC (USA) | Atmospheric pressure index |
| IOD / DMI | NOAA PSL / JAMSTEC | Ocean temperature index |
| Rainfall anomalies | CHIRPS v3 / UC Santa Barbara | Gridded satellite + gauge rainfall |
| Reservoir storage | India-WRIS / CWC (India) | Aggregated water storage volumes |

None of this data is linked to individuals. It is purely geophysical and climatological in nature.

## Children's Privacy

This site does not knowingly collect data from anyone, including children under 13 (COPPA) or under 16 (GDPR). Since no data is collected from anyone, no special provisions are required.

## GDPR / CCPA Compliance

Because **no personal data is collected or processed**, this dashboard has no GDPR or CCPA obligations. There is no data to access, correct, delete, or port.
